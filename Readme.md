# Customer Churn Prediction

## Overview
This project predicts whether a customer is likely to leave a service.
It also explains *why* the model made a prediction using simple visual explanations.

## Problem
Customer churn reduces revenue.
Businesses need not only predictions, but also clear reasons behind them to take action.

## What This Project Does
- Takes customer details as input
- Predicts whether the customer will churn
- Shows which factors influenced the prediction

## How It Works (High Level)
- A trained machine learning model predicts churn
- SHAP is used to explain the model’s output
- Results and explanations are displayed in a simple interface

## Tech Used
- Python
- Machine Learning
- Streamlit
- SHAP (for model explanation)

## Use Case
- Customer retention analysis
- Understanding key churn drivers
- Learning project for interpretable machine learning

## Status
Working prototype with prediction and explanation support.
